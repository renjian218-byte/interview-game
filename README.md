# 就活面接練習ゲーム（interview-game）

ブラウザだけで動く、就職活動の面接対策ゲームです。ナビゲーター「櫻みゆき」と一緒に、面接マナー・受け答え・敬語・就活用語をゲーム感覚で練習できます。選択肢を選んで進めると、面接官の表情とコメントで回答の好印象度が即座に分かり、プレイのたびに経験値（EXP）が貯まってレベルが上がる育成要素つきです。

# 就活面接練習ゲーム（interview-game）

ブラウザだけで動く、就職活動の面接対策ゲームです。ナビゲーター「櫻みゆき」と一緒に、面接マナー・受け答え・敬語・就活用語をゲーム感覚で練習できます。選択肢を選んで進めると、面接官の表情とコメントで回答の好印象度が即座に分かり、プレイのたびに経験値（EXP）が貯まってレベルが上がる育成要素つきです。

## 概要

- **対象**：これから面接を控える就活生・転職活動中の人
- **できること**：入室から退出までの基本マナー、自己PR・志望動機の受け答え、敬語、就活用語までを、選択式で実践的に練習
- **特徴**：面接官の表情とフィードバックで「今の答えが好印象かどうか」が直感的に分かる

## 画面の流れ

スタート画面（メインハブ） → モード選択 → プレイ画面 → 結果画面

メインハブの上部にプレイヤーレベル・経験値・クリア回数のステータスバー、下部に「面接」「トレーニング」の2モードへの入口を配置しています。

## 主な機能

### 面接モード

| 難易度 | テーマ | 問題数 |
| --- | --- | --- |
| 初級 | 入室から退出までの基本マナー | 10問 |
| 中級 | 自己PR・志望動機・深掘り質問 | 10問 |
| 上級 | 最終面接・圧迫気味の質問・価値観の深掘り | 10問 |
| ケース面接 | 対話で進める構造化・仮説立て・施策提案 | 8シナリオ |

各選択肢に「正解／普通／少し悪い／かなり悪い」のランクと配点を設定。選択するとキャラの表情が変化し、面接官コメントと解説フィードバックが表示されます。ケース面接は「最初の問い → 追加情報 → 次の問い」と対話が2段階で進む構造です。

### トレーニングモード

- **敬語クイズ**：尊敬語・謙譲語・丁寧語の3カテゴリ。各20問のプールから毎回ランダムで10問出題。各カテゴリに解説ガイド付き。
- **就活専門用語**：初級・中級・上級の3難易度。8問以上正解でクリアし、次の難易度がアンロックされる進行制。

### 育成・評価システム

- 合計点の達成率で **S・A・B・C・D** の5段階判定
- 結果に応じてEXPを獲得し、100EXPごとにレベルアップ
- レベル・EXP・クリア回数をステータスバーに反映

## 技術構成

- **フロントエンド**：HTML / CSS / 素のJavaScript（ビルド・外部ライブラリ・サーバー処理なし）
- **データ保存**：ブラウザの localStorage（レベル・EXP・クリア回数・最高スコア・難易度解放状況）
- **ホスティング**：Vercel
- **対応**：スマホ表示前提のレイアウト。apple-touch-icon によりホーム画面追加に対応

### ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面構造 |
| `style.css` | デザイン |
| `script.js` | ゲームロジック・進行管理 |
| `keigo-quiz-data.js` | 敬語クイズの問題データ |
| `shukatsu-terms-data.js` | 就活用語の問題データ |
| `images/` `assets/` | 画像素材（キャラ表情差分・背景など） |

## 設計のポイント

- **データとロジックの分離**：問題データを専用JSファイルに切り出し、内容の追加・編集をしやすい構造にした
- **出題のランダム化**：問題・選択肢をシャッフルし、繰り返しプレイでも新鮮さを維持
- **難易度アンロック制**：段階的な達成感を設計
- **フォールバック設計**：画像が読み込めない環境でも操作を継続できる
- **外部依存ゼロ**：ブラウザだけで完結するため、配布・公開が容易
- **画面幅の最適化**：スマホ前提のUIだが、PCで開いた際もモバイル幅のコンテナを中央寄せで表示し、レイアウトが崩れないようにした

## ローカルでの動かし方

リポジトリをクローンし、`index.html` をブラウザで開くだけで動作します。

```bash
git clone https://github.com/renjian218-byte/interview-game.git
cd interview-game
# index.html をブラウザで開く
```
